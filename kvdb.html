Key-value database¶

Accessing through the Context class¶

The octobot.Context provides quick “shortcut” to per-user (octobot.Context.user_db) and per-chat (octobot.Context.chat_db) databases

Accessing manually¶

The key-value database can be accessed manually, per chat_id:

import octobot

octobot.Database[777000]["key"] = "value"

print(octobot.Database[777000]["key"])

Key-value database classes¶

class octobot.database.RedisData(redis_db, chat_id)¶

Redis chat/user data class. Can be accessed like dictionary.

get(key, default=None, dont_decode=False)¶

Gets key from database. If database cant be accessed or key cant be found, will return value of default param

Parameters

  • item (str) – Key to lookup

  • default – “Default” value

  • dont_decode (bool) – If value should be decoded from bytes or not. Defaults to False

Returns

The lookup results.

set(key, value)¶

Sets key to value

Parameters

  • key (str) – Key to set

  • value (any) – Value

Raises

octobot.exceptions.DatabaseNotAvailable if database is not accessible

Returns
