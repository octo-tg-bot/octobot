Permission decorators¶

octobot.permissions(**perms_kwargs)¶

Decorator to check permissions.

Parameters

perms – Valid permissions that can be found on telegram.ChatMember + is_admin. Example: @permissions(can_delete_messages=True). The value does not matter.

octobot.my_permissions(**perms_kwargs)¶

Decorator to check bots own permissions.

Parameters

perms – Valid permissions that can be found on telegram.ChatMember + is_admin. Example: @my_permissions(can_delete_messages=True). The value does not matter.

Decorator to check if sender is not an admin.
