PluginInfo class¶

class octobot.PluginInfo(name: str, reply_kwargs: dict = <factory>, handler_kwargs: dict = <factory>, after_load: Callable[[octobot.OctoBot], Any] = None, state: octobot.enums.PluginStates = <PluginStates.unknown: 1>, state_description: str = 'state_description was not overwritten', last_warning: str = None, can_disable: bool = True)¶

Plugin info class

Parameters

  • name (str) – Name of plugin that will show up in /help. The only required argument

  • reply_kwargs (dict, optional) – Keyword arguments that will be passed to all context.reply calls inside plugin

  • handler_kwargs – Keyword arguments that will be passed to all handlers inside the plugin

  • after_load (callable, optional) – Function that will be called after bot finishes loading all plugins

  • state (octobot.enums.PluginState, optional) – Plugin state. Will be overwritten by loader if set to default (PluginStates.unknown). Will also be overwritten if set to default and logger.warning method was used

  • state_description (str, optional) – Description for current plugin state

  • module – Plugin module. Dont pass anything to that argument, used by loader and handlers

  • can_disable (bool) – If plugin can be disabled using /disable_plugin. All plugins located in base_plugins cant be disabled no matter if variable set to false or not.

Variables

logger – Logger, generated by PluginInfo. Not an argument, a variable
